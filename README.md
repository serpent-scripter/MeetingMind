# MeetingMind

MeetingMind is a web application designed to help you transcribe meetings and automatically extract actionable insights and action items.

## Architecture

- **Frontend:** Built with Next.js for a responsive and modern user interface.
- **Backend:** Built with FastAPI, handling meeting audio processing, transcriptions, action item extraction, and serving API requests.

## Getting Started

Please refer to the `frontend/README.md` and `backend/README.md` for specific instructions on setting up and running each component of the application.
